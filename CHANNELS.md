# Channel Isolation — безопасное разделение каналов

## Модель доступа

| Канал | Доступ | Шифрование | Макс пиров |
|-------|--------|-----------|------------|
| `chat` | 👥 Group (по токену) | ❌ | 100 |
| `findings` | 👥 Group (по токену) | ❌ | 100 |
| `voice` | 🌐 Public | ❌ | 6 |
| `admin` | 🔐 PeerList | ✅ | 3 |

## Команды

```
/channel create <name> — создать канал
/channel allow <name> <peer> — разрешить пиру доступ
/channel deny <name> <peer> — запретить
/channel list — список каналов
```

## Уровни доступа

```
🌐 Public    — любой пир в сети
👥 Group     — по токену группы
🔐 PeerList  — только разрешённые пиры
🔒 Private   — только локально
```
