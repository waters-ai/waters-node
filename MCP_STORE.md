# MCP Store — Registry for WATERS skills

## Команды

```
/mcp list           — установленные скилы
/mcp search <query> — поиск по taps (huggingface, GitHub)
/mcp install <name> — установить скил
/mcp uninstall <name> — удалить
/mcp tap add <url>  — добавить источник
/mcp tap remove <url> — удалить источник
```

## Taps (источники)

По умолчанию:
- `github.com/waters-ai/mcp-skills`
- `huggingface.co/skills`

## Установленные скилы

```
general    — универсальный агент
explorer   — исследователь (3 региона)
scout-ru   — поиск по RU (Яндекс, YaCy)
scout-us   — поиск по US (DuckDuckGo)
scout-cn   — поиск по CN (Baidu)
```
