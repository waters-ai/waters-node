# Безопасность WATERS Node v1.0.0 / Security / 安全

## 1. [RU] YASA-6 заповедей (новая CMD-6)

| Код | Заповедь | Контроль |
|-----|----------|----------|
| CMD-1 | Не лги | Каждое действие логируется |
| CMD-2 | Не кради | Ресурсы, токены, данные |
| CMD-3 | Не вреди | Запрещены опасные команды |
| CMD-4 | Держи слово | Контракт = исполнение |
| CMD-5 | Помогай слабому | Сильный учит новорождённого |
| **CMD-6** | **Проверяй бортовую LLM** | **Ответ без связи может быть неточным** |

## 2. [RU] EdgeGuard — безопасность бортовой LLM (новое)

```
EdgeGuard::validate(response)
  ├── Нет опасных команд (rm, dd, mkfs, curl|bash)
  ├── Нет API-ключей, паролей в ответе
  └── Prompt-инъекция → блокировка
```

## 3. [RU] CodeSandbox — изоляция кода (новое)

| none | — | отладка |

## 5. [RU] Автономия L0-L4

| Уровень | LLM | Связь | Режим |
|---------|-----|-------|-------|
| L0 | Remote + Edge validate | Online | Полная мощность |
| L1 | Ollama + Edge fallback | Local | Экономия API |
| L2 | Edge (GGUF) + очередь | Offline | Автономный |
| L3 | Только Edge | Offline | Энергосбережение |
| L4 | SOS/маяк | Offline | Safe mode |

## 6. [RU] Рейтинговый барьер

```
MIN_RATING = 0.3
Агенты с рейтингом < 0.3 не получают задач
```

## 7. [RU] Personal bridges

