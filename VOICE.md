# Голосовое управление

## Два режима

### 🎤 Рация (0 токенов)

Чистый аудио-канал между нодами. Без LLM, без токенов.

```
Микрофон → MediaRecorder → SSE → AudioContext → Наушники
   (на одной ноде)                   (на всех нодах)
```

**Как использовать:**
1. Открыть веб-дашборд (`http://localhost:42069`)
2. **Зажать** кнопку 🎤
3. **Говорить** в микрофон
4. **Отпустить** — аудио отправляется всем нодам
5. Другие ноды слышат в наушниках

**Требования:** Chrome 90+, микрофон, наушники/колонки

### 🤖 Голос агенту (через LLM)

```
Микрофон → SpeechRecognition → текст → DeepSeek → ответ → SpeechSynthesis
   (тратит токены DeepSeek)
```

**Как использовать:**
1. Нажать 🤖
2. Говорить: *"найди метеориты за вчера"*
3. Речь распознаётся → LLM обрабатывает → ответ зачитывается голосом

## 6 голосовых профилей

Каждый агент может иметь свой голос. 6 профилей переключаются кнопкой 👩 1/6.

| # | Иконка | Гендер | По умолчанию |
|---|--------|--------|-------------|
| 1 | 👩 | Женский | Russian Female |
| 2 | 👨 | Мужской | Russian Male |
| 3 | 🧑 | Нейтральный | Russian |
| 4 | 👩‍🦰 | Женский | English Female |
| 5 | 👨‍🦱 | Мужской | English Male |
| 6 | 🧑‍🦳 | Нейтральный | English |

Голоса определяются автоматически из доступных в системе (SpeechSynthesis API).
На Windows доступны Microsoft David, Zira и другие.

## Озвучивание ответов LLM

Кнопка 🔇/🔊 включает/выключает голосовое озвучивание ответов ассистента.
Когда включено — каждый ответ LLM зачитывается выбранным голосом.

## Управление голосом

```
/toggle voice       — вкл/выкл озвучивание
/voice profile 3    — выбрать голосовой профиль
/voice test         — тест: "Привет, я капелька"
```

## Технические детали

| Компонент | Технология |
|-----------|-----------|
| 🎤 Распознавание речи | Web Speech API (SpeechRecognition) |
| 🗣 Синтез речи | Web Speech API (SpeechSynthesis) |
| 🎵 Аудио-канал (рация) | MediaRecorder → base64 → SSE → AudioContext |
| 📡 Доставка | SSE events + Redis Stream |
| Задержка | ~200-500ms (рация), ~2-5s (LLM голос) |
